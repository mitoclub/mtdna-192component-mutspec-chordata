3. C is more often abasic

AB$AbasicSiteOnHeavyStrand = ''
for (i in 1:nrow(AB))
{ # i = 1
AB$AbasicSiteOnHeavyStrand[i] = unlist(strsplit(AB$origtriH[i],''))[2]
}
